# FE Classroom Quiz

基本情報技術者試験の C# プログラミング演習クイズ。C# コードと擬似言語（基本情報試験形式）の比較学習に対応しています。

## 使い方

### 1. 通常の利用（自習・先生）

1. ページを開く（例：`https://xxx.github.io/fe-learning-lab/`）
2. **固定プリセット**（初級・標準・応用・条件式のみ・配列のみ・基礎ミックス）のいずれかをクリック
3. **出題設定**を調整（任意）
   - 初心者モード：日本語化・補足を表示
   - 問題数：5 / 10 / 15 / 20 / 全件
   - 難易度：★〜★★★ のチェック
   - 言語・ジャンル：出題範囲を指定
4. **「開始」** をクリック
5. 問題を解く → **「採点する」** → **「解説を見る」** で擬似言語を確認 → **「次へ」**

### 2. 先生用（プリセット保存・配布）

URL に `?teacher=1` を付けて開く

- **保存プリセット**：現在の設定をブラウザに保存（後から呼び出し可能）
- **プリセットをダウンロード**：JSON をダウンロードし、`presets/` に置いて配布可能
- **Export/Import**：プリセットを JSON で出力・取り込み

### 3. 生徒用リンク（授業・配布）

| 用途                       | リンク例                                                             |
| -------------------------- | -------------------------------------------------------------------- |
| 授業（最初から・自動開始） | `?student=1&preset=conditionsOnly&fresh=1&autostart=1&qver=20260209` |
| 自宅復習（続きから可）     | `?student=1&preset=conditionsOnly&autostart=1&qver=20260209`         |

生徒用は設定変更不可・タブ1つのみ利用を推奨。

## パラメータ

| パラメータ  | 説明                                                |
| ----------- | --------------------------------------------------- |
| student=1   | 生徒ロック（設定変更不可）                          |
| teacher=1   | 先生用UIを表示                                      |
| preset=XXX  | 強制プリセット（presets/ のファイル名、拡張子なし） |
| autostart=1 | 起動直後に開始                                      |
| fresh=1     | 必ず最初から（保存セッション破棄）                  |
| qver=...    | questions.json のキャッシュ対策                     |

## プリセットの仕組み

`preset=XXX` の解決順序：
1. **presets/XXX.json**（プロジェクト内のファイル）
2. localStorage の保存プリセット
3. 固定プリセット（beginner 等）
4. フォールバック：beginner

### 固定プリセット一覧

| プリセット                  | 内容                                              |
| --------------------------- | ------------------------------------------------- |
| 初級 (beginner)             | 条件式のみ・★・10問・初心者モードON               |
| 標準 (standard)             | 条件式+ループ+配列・★★・10問                      |
| 応用 (advanced)             | 条件式+ループ+配列・★★★・10問                     |
| 条件式のみ (conditionsOnly) | 条件式のみ・★〜★★★・10問                          |
| 配列のみ (arraysOnly)       | 配列のみ・★〜★★★・10問                            |
| 基礎ミックス (mixBasics)    | 易しい問題を複数ジャンル・★・10問・初心者モードON |

## 先生による追加プリセットの配布

1. 出題設定（問題数・難易度・言語・ジャンル等）を画面上で設定
2. プリセット名を入力（例：`lesson3`）
3. 「プリセットをダウンロード」をクリック → `lesson3.json` がダウンロードされる
4. リポジトリの `presets/` フォルダにファイルを配置
5. コミット・プッシュ
6. 生徒へリンクを共有：`?student=1&preset=lesson3&autostart=1`
